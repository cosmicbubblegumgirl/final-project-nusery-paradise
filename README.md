# Paradise Nursery | Pixie Petal Patch Final Project

This is a **React + Redux Toolkit + Vite** shopping cart application built for the **Paradise Nursery** final project, while preserving the whimsical **Pixie Petal Patch** nursery theme.

## What is included

- Landing page with the company name, about section, and **Get Started** button
- Background image implemented in `src/App.css`
- Plant product listing grouped into **4 categories**
- **6 unique plants per category** using the plant images you provided
- Add to Cart button that:
  - adds the item to the shopping cart
  - disables after the item is added
  - updates the cart count in the navbar
- Navbar links for **Home**, **Plants**, and **Cart**
- Shopping cart page with:
  - item thumbnail, name, and unit price
  - increase quantity button
  - decrease quantity button
  - delete button
  - total per plant
  - total number of plants
  - grand total cost
  - checkout button with a **Coming Soon** message
  - continue shopping button

## Run locally

```bash
npm install
npm run dev
```

Then open the local Vite link shown in the terminal.

## Build for production

```bash
npm run build
```

## Upload to GitHub

1. Create a **public** GitHub repository.
2. Upload all files from this project.
3. Commit and push to the `main` branch.
4. Open each required file on GitHub.
5. Copy the URL of each file and paste it into the grader.
